# Артур и Алиса — статическая копия приглашения

В архиве находится автономная копия сайта-приглашения: `index.html`, `styles.css`, `script.js`.

## Что заменено

- Все фото и видео заменены на CSS-заглушки.
- Музыка заменена на демо-кнопку состояния, без аудиофайла.
- Анкета работает в демо-режиме: ответы сохраняются в `localStorage` браузера, без отправки на сервер/Tilda.

## Как открыть

Откройте `index.html` в браузере или загрузите все файлы в одну папку на хостинг/GitHub Pages.

## Что реализовано

- Mobile-first свадебное приглашение.
- Заставка загрузки.
- Главный экран с именами, датой и кнопкой музыки.
- Блок приглашения, история, календарь, локация, программа дня, дресс-код, второй день, предупреждение и анкета гостя.
- Анимации появления при прокрутке, прогресс прокрутки, плавный параллакс главного медиа.
- Desktop-сообщение: «Пожалуйста, откройте сайт через мобильное устройство».
